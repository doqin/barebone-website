Bao

Bao is a programming language with Vietnamese syntax meant to be educational to code beginners. The language is planned to be extensible with C/C++ libraries. The language is inspired by many high-level languages like Pascal and Go designed to be simple and safe but fast enough to cater to many types of projects.

  • Software design — System programming — Architecture
  • C++ – LLVM – LLD
