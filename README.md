# Prerequisites
- Windows SDK requried — install [here](https://developer.microsoft.com/en-us/windows/downloads/windows-sdk/)
- MinGW — follow the tutorial [here](https://code.visualstudio.com/docs/languages/cpp)

# Building
Create a `tasks.json` file inside the `.vscode` folder in the working directory and use a configuration similar to this
```json
{
    "tasks": [
        {
            "type": "cppbuild",
            "label": "C/C++: g++.exe build active file",
            "command": "g++",
            "args": [
                "-fdiagnostics-color=always",
                "-g",
                "${cwd}\\main.cpp",
                "${cwd}\\src\\httpbuilder.cpp",
                "${cwd}\\src\\utils.cpp",
                "-o",
                "${cwd}\\build\\output.exe",
                "-lws2_32",
                "-I${cwd}\\include",
            ],
            "problemMatcher": [
                "$gcc"
            ],
            "group": {
                "kind": "build",
                "isDefault": true
            },
            "detail": "Task generated by Debugger."
        }
    ],
    "version": "2.0.0"
}
```
For C/C++ intellisense, consider creating a `c_cpp_properties.json` file inside of `.vscode` and use a similar configuration like below
```json
{
    "configurations": [
        {
            "name": "Win32",
            "includePath": [
                "${workspaceFolder}/**",
                "${workspaceFolder}",
                "C:\\msys64\\ucrt64\\include\\**", // Change this if you use a different compiler
                "C:\\Program Files (x86)\\Windows Kits\\10\\Include\\10.0.26100.0\\um",
                "C:\\Program Files (x86)\\Windows Kits\\10\\Include\\10.0.26100.0\\shared" // Make sure to install the Windows SDK for these headers
            ],
            "defines": [
                "_DEBUG",
                "UNICODE",
                "_UNICODE"
            ],
            "cStandard": "c11",
            "cppStandard": "c++17",
            "compilerPath": "C:\\msys64\\ucrt64\\bin\\g++.exe" // Change this if you use a different compiler
        }
    ],
    "version": 4
}
```
Click the run C/C++ file button on the top right of the tab section to run the project

Access the website at `localhost:2452`
